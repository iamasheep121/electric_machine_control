## 一、电机的主要控制参数（细化和补充）

| 参数     | 物理意义          | 单位              | 如何控制           | 难点                       |
| ------ | ------------- | --------------- | -------------- | ------------------------ |
| **速度** | 转子旋转的快慢       | rpm（转/分）或 rad/s | 调电压/PWM占空比、调电流 | 低速稳定性、高速弱磁、转速波动抑制        |
| **位置** | 转子停止的角度或转过的圈数 | 度、圈、rad         | 闭环PID、编码器反馈    | 定位精度、超调、稳态误差、回零策略        |
| **扭矩** | 电机输出的旋转力      | N·m（牛顿·米）       | 控制电流（扭矩正比于电流）  | 扭矩波动、过载能力、启动扭矩、恒扭矩区/恒功率区 |

> 三个参数相互耦合：调速度会影响扭矩，调位置会影响速度。好的控制算法要解耦。

## 二、直流有刷电机（Brushed DC, BDC）—— 最基础的电机

### 工作原理

- 定子是永磁体或励磁绕组，转子是绕组+换向器
- 电刷与换向器接触，改变电流方向，使转子持续旋转

### 控制方式

- **调速**：改变电压/PWM占空比（电压越高越快）
- **换向**：由电刷和换向器自动完成，控制器无需关心换向
- **正反转**：改变电压极性

### 硬件驱动

- 最简单的：一个MOSFET + 二极管（只能单向，不可反转）
- 标准方案：**H桥电路**（4个MOSFET，可正反转、刹车、滑行）

### 优点

- 控制极简：给电就转，调压调速
- 成本低
- 启动力矩大

### 缺点

- 电刷磨损 → 寿命短（几百到几千小时）
- 产生电火花 → 电磁干扰
- 高速下电刷摩擦大，效率下降
- 需要定期维护（更换电刷）

### 典型应用

- 电动工具（电钻、角磨机，低端）
- 玩具小车
- 汽车雨刮器、车窗升降
- 便宜的电动自行车

### 选型参数

- 额定电压（3V-48V常见）
- 空载转速（rpm）
- 额定扭矩/堵转扭矩
- 额定电流/堵转电流（选MOSFET的关键）

## 三、无刷电机（细分详解）

### 3.1 共同原理

- 定子是绕组（线圈），转子是永磁体
- 控制器按顺序给不同绕组通电，产生旋转磁场，带着永磁转子转
- **关键**：需要知道转子位置才能正确换相

### 3.2 转子位置检测方式

| 方式        | 原理              | 优点       | 缺点             | 成本    |
| --------- | --------------- | -------- | -------------- | ----- |
| **霍尔传感器** | 三个霍尔元件检测磁场变化    | 简单、可靠、便宜 | 增加布线、传感器可能损坏   | 低     |
| **无传感器**  | 检测反电动势（BEMF）过零点 | 无线缆、更可靠  | 低速性能差、启动需要特殊算法 | 无额外成本 |

### 3.3 BLDC（无刷直流电机，方波驱动）

#### 控制方式

- **六步换相法**：每60度电角度换相一次，每次两相通电
- 驱动波形：方波（实际是梯形波）
- 控制量：PWM占空比 → 平均电压 → 转速

#### 优点

- 控制简单（换相逻辑清晰）
- 无电刷，寿命长，维护少
- 效率较高

#### 缺点

- 扭矩波动明显（换相瞬间）
- 低速运行不平稳
- 噪声比PMSM大

#### 典型应用

- 航模无人机、电调（ESC）
- 电动车（早期、低端）
- 风扇、水泵
- 电动工具（中高端）

### 3.4 PMSM（永磁同步电机，正弦波驱动）

#### 控制方式

- 需要**FOC（磁场定向控制）**算法
- 驱动波形：正弦波（实际是SVPWM合成）
- 控制量：d轴电流（励磁）、q轴电流（扭矩）

#### 优点

- **扭矩波动极小**（几乎没有）
- 运行安静、平滑
- 效率高（尤其低速大扭矩）
- 响应快，精度高

#### 缺点

- 控制复杂（需要数学变换：Clark/Park、PID、SVPWM）
- 计算量大（需要较强MCU）
- 调试难度高

#### 典型应用

- 电动车（主流）
- 工业伺服
- 机器人关节
- 高端家电（空调、洗衣机、冰箱压缩机）

### 3.5 BLDC vs PMSM 对比总结

| 维度      | BLDC（方波） | PMSM（正弦波/FOC） |
| ------- | -------- | ------------- |
| 驱动波形    | 方波/梯形波   | 正弦波           |
| 控制复杂度   | 低（六步换相）  | 高（FOC）        |
| 扭矩波动    | 明显       | 极小            |
| 噪声      | 大        | 小             |
| 效率      | 中高       | 高             |
| 低速性能    | 一般       | 优秀            |
| 成本（控制器） | 低        | 高             |
| MCU要求   | 普通8位/32位 | 带FPU/数学加速的32位 |

## 四、步进电机（详细版）

### 工作原理

- 定子有多相绕组（通常2相），转子是永磁/软磁齿状结构
- 按顺序给各相通电，转子一步一停地旋转
- **步距角**：一个脉冲转动的角度，常见1.8°（200步/圈）、0.9°（400步/圈）

### 控制方式

- **脉冲+方向**（最常用）
  - 一个脉冲 → 转一步
  - 方向信号 → 正转/反转
- **细分驱动**：通过控制各相电流比例，把一步分成更小的微步（16细分、32细分等），降低低频振动

### 硬件驱动

- 专用驱动芯片：A4988、DRV8825、TMC2209（静音）
- 主控只需要输出脉冲信号，驱动芯片负责电流控制

### 优点

- **开环控制即可定位**（不需要编码器）
- 价格便宜
- 停止时有保持扭矩（自锁）
- 结构简单，可靠性高

### 缺点

- 转速不能太快（通常不超过1000rpm）
- 高速扭矩急剧下降
- 可能**失步**（负载过大或加速过快时丢步）
- 效率低，发热大
- 噪音和振动（尤其低速时）

### 失步的解决方法

- 加编码器做闭环（闭环步进，成本介于步进和伺服之间）
- 降低加速度
- 使用更大扭矩的步进电机
- 使用细分驱动

### 典型应用

- 3D打印机
- 雕刻机/激光切割机
- 相机滑台、电动显微镜
- 自动门锁
- 安防球机

### 选型参数

- 步距角
- 保持扭矩（N·m或kg·cm）
- 相电流
- 相电阻/相电感
- 机身尺寸（42步进、57步进等）

## 五、舵机（详细版）

### 工作原理

- 内部：直流有刷电机 + 减速齿轮组 + 控制电路 + 电位器（位置反馈）
- 闭环控制：MCU发PWM信号 → 控制电路比较目标角度和当前角度 → 驱动电机转 → 到达目标角度后停止

### 控制信号（标准）

- PWM周期：20ms（50Hz）
- 脉冲宽度对应角度：
  - 0.5ms → 0°
  - 1.5ms → 90°
  - 2.5ms → 180°
- 不同品牌略有差异，看数据手册

### 优点

- 集成度高：一个模块包含电机、减速、反馈、驱动
- 控制简单：一根信号线，给PWM就转
- 即插即用，方便快速原型

### 缺点

- 只能转有限角度（通常0-180°，少数0-270°）
- 不能连续旋转（除非改装）
- 精度和响应速度有限（不如工业伺服）
- 内部电位器会磨损（寿命不如无刷舵机）
- 普通舵机有死区（小角度信号不响应）

### 价格

- 9g塑料舵机：5-15元
- 标准金属齿舵机：30-80元
- 高压/大扭矩/无刷舵机：100-500元

### 扩展：连续旋转舵机

- 改装或购买专门型号
- 控制信号决定转速和方向，不是角度
- 相当于把舵机当有刷电机用（不推荐，性价比低）

### 典型应用

- 遥控小车转向
- 机器人关节（简易机器人）
- 机械臂（学习用）
- 相机云台（低端）

## 六、伺服电机（工业级，补充）

> 注意：日常说的“舵机”也叫servo，但工业伺服是完全不同的东西

### 组成

- 电机本体（通常是无刷/永磁同步）
- 编码器（多圈绝对值、光电、磁编码器等，分辨率很高）
- 伺服驱动器（负责FOC控制、位置/速度/扭矩环）

### 控制方式

- **位置模式**：上位机发目标位置，驱动器完成定位
- **速度模式**：上位机发目标速度
- **扭矩模式**：上位机发目标电流/扭矩
- 控制接口：脉冲方向、模拟量、CANopen、EtherCAT、Modbus等

### 优点

- 精度极高（编码器分辨率高，定位准）
- 响应快（带宽高）
- 扭矩大、过载能力强
- 运行平滑
- 支持各种高级功能（陷波滤波器、振动抑制、在线自整定）

### 缺点

- 贵（电机+驱动器通常是步进的数倍到数十倍）
- 配置复杂（要调PID参数）
- 需要编码器反馈（增加线缆）

### 典型应用

- CNC数控机床
- 工业机器人
- 自动化生产线
- 高端3D打印机
- 医疗设备

## 七、电机控制的核心组件

### 控制器（大脑）

| 类型       | 例子                     | 适用场景            |
| -------- | ---------------------- | --------------- |
| 简单MCU    | STM32F103、Arduino      | BLDC六步换相、步进脉冲控制 |
| 带FPU/MCU | STM32F4/F7/G4、TI C2000 | PMSM、FOC        |
| 专用驱动芯片   | MCP8024、DRV8301        | 集成MOSFET预驱，简化设计 |
| 一体化驱动器   | ODrive、SimpleFOC       | 适合快速原型          |

### 功率级（肌肉）

- MOSFET/IGBT：开关器件，负责给电机供电
- 预驱（Gate Driver）：放大MCU的PWM信号，驱动MOSFET
- 电流采样：采样电阻 + 运放，用于电流环控制

### 反馈（感官）

- 霍尔传感器：位置/速度（低精度）
- 编码器（增量/绝对值）：高精度位置/速度
- 旋转变压器：极高精度，耐恶劣环境

## 八、一个完整的电机控制系统框图

```
上位机（目标：速度/位置/扭矩）
    ↓
MCU（执行控制算法：PID、FOC、换相逻辑）
    ↓ 输出PWM
预驱（Gate Driver）
    ↓ 驱动信号
MOSFET H桥/三相桥
    ↓ 电压/电流
电机（转动）
    ↓ 带动
负载
    ↑ 反馈信号
传感器（霍尔/编码器/电流采样）
    ↑ 回到MCU
```

## 九、学习顺序建议（由浅入深）

1. **直流有刷电机**：H桥、PWM调速、正反转 → 半小时搞定
2. **舵机**：PWM控制角度 → 10分钟搞定
3. **步进电机**：脉冲控制、细分驱动 → 半天搞定
4. **BLDC（六步换相）**：霍尔传感器 + 六步换相逻辑 → 1-2周
5. **PMSM（FOC）**：Clark/Park、PID、SVPWM → 1-3个月

## 十、常用术语速查表

| 术语      | 含义                     |
| ------- | ---------------------- |
| PWM     | 脉宽调制，控制平均电压/电流         |
| 占空比     | PWM高电平时间占总周期的比例，0-100% |
| H桥      | 驱动有刷电机/步进电机一相的电路       |
| 六步换相    | BLDC的换相顺序，共6个状态        |
| 反电动势    | 电机转动时产生的反向电压，用于无传感器检测  |
| FOC     | 磁场定向控制，PMSM的主流算法       |
| Clark变换 | 三相电流 → 两相静止坐标系         |
| Park变换  | 两相静止坐标系 → 两相旋转坐标系      |
| SVPWM   | 空间矢量PWM，生成正弦波驱动        |
| PID     | 比例-积分-微分控制器，闭环控制基础     |
